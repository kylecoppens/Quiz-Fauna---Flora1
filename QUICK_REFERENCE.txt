📸 IMAGE SOURCES

  ✓ High resolution available (640px+ minimum)
  ✓ Species identification verified
  ✓ URLs tested and working

To find more images:
  1. Visit https://commons.wikimedia.org/
  2. Search scientific name (e.g., "Turdus merula")
  3. Filter by CC-BY-SA license
  4. Copy thumbnail URL in format:
     https://upload.wikimedia.org/wikipedia/commons/thumb/.../640px-...
